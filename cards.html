главная о проекте

Современный кроссплатформенный API для высокопроизводительной 3D-графики и вычислений. Предоставляет низкоуровневый контроль над GPU, значительно снижая нагрузку на CPU по сравнению с традиционными API.

Кроссплатформенный API для рендеринга 2D и 3D-графики. Обеспечивает аппаратную независимость и стандартизированный интерфейс для работы с графическими процессорами.

Набор API от Microsoft для мультимедийных приложений, особенно игр. Включает компоненты для графики, звука, ввода и сетевого взаимодействия. Доминирующая технология для разработки игр на платформе Windows.
