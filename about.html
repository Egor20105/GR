О ПРОЕКТЕ

Данный веб-сайт был создан Егором Балакиным для демонстрации в компьютерном классе. Проект представляет собой мини-портфолио современных технологий компьютерной графики и включает:

  • Навигационную панель
  • Интерактивую карусель
  • Информационные карточки о популярных графических API

Технологии: HTML5, CSS3, Bootstrap 5
